# Prefect Agents

## What it is
Workflow orchestration platform that can host agentic tasks as durable, observable flows. Agents are embedded inside Prefect flows.

## Distinctive traits
- Native observability and state persistence from Prefect.
- DAG-based orchestration with retries and caching.
- Cloud and self-hosted deployment options.
