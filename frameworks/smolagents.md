# Smolagents

## What it is
A lightweight, code-first agent framework from Hugging Face. Agents write Python code snippets to use tools rather than JSON tool schemas.

## Distinctive traits
- Code-based tool execution by default.
- Minimal API surface; easy to extend with custom tools and models.
- Targets experimentation and quick prototyping.
