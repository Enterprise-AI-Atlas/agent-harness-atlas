# Aider

## What it is
A terminal-based AI pair programming tool. Aider composes code edits from natural language requests using git-aware context.

## Distinctive traits
- Deep git integration: reads diffs, creates commits, manages branches.
- Multi-file editing with code-map context.
- Supports many LLM providers via LiteLLM.
