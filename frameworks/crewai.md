# CrewAI

## What it is
A framework for building role-based agent "crews" that work together on tasks. Each agent has a role, goal, backstory, and delegated tasks.

## Distinctive traits
- Strong emphasis on role definitions and structured task delegation.
- Sequential, hierarchical, and process-based crew execution models.
- Memory, guardrails, and tool use layered on top of agent roles.
