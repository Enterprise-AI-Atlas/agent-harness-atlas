# Claude Code

## What it is
Anthropic's agentic coding assistant, distributed as a CLI. Reads codebases, edits files, runs commands, and explains changes.

## Distinctive traits
- Strong codebase comprehension via tools and context windows.
- Approval prompts before shell commands and file edits.
- Tied to Anthropic's Claude models.
