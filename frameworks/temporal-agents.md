# Temporal Agents

## What it is
Temporal's durable execution platform applied to agentic workflows. Agents are Temporal workflows/activities that survive failures and retries.

## Distinctive traits
- Durable execution with strong guarantees around state and retries.
- State-machine-friendly long-running agent processes.
- Multi-language SDKs (TypeScript, Python, Go, Java, .NET).
