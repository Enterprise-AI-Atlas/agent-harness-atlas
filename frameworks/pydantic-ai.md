# Pydantic AI

## What it is
A Python agent framework built by the Pydantic team. Agents are strongly typed and results are validated with Pydantic models.

## Distinctive traits
- Pydantic-typed inputs/outputs and dependency injection.
- Supports multi-turn tool calling with validated structured results.
- Built-in tracing and metrics via `logfire` integration.
