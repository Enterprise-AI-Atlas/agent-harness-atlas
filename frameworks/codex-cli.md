# Codex CLI

## What it is
OpenAI's agentic coding CLI. Performs terminal commands, file edits, and other software engineering tasks with human approval for risky actions.

## Distinctive traits
- Tight integration with OpenAI models (o3, o4-mini, GPT-4.1).
- Built-in approval modes for commands and file changes.
- Designed for safe, autonomous coding workflows.
