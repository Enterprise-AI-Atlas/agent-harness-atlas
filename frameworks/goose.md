# Goose

## What it is
An open-source, extensible AI agent from Block. Runs locally and can be extended with custom extensions for different tools and environments.

## Distinctive traits
- Rust core with TypeScript extension API.
- MCP-native extension model.
- Desktop and CLI modes; locally run.
